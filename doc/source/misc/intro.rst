Einführung
==========
pyCommons bietet Module, um immer wieder benötigte Python-Algorithmen zu vereinfachen. 
In vielen Applikationen werden folgende Module immer wieder benötigt:

 - Datumsfunktionen.
 - Zugriff auf einfache Dateiformate.
 - Baumaufbau mit ``list_of_dictionarys``.
 
pyCommons vereinfacht zudem den Zugriff auf viele Dateiformate, die bereits durch die
Standard-Bibliothek von Python bereitgestellt werden. Angestrebt wird vor allem ein
vereinheitlichtes Handling der verschiedenen Dateiformate.

Diverse Module sind abhängig von anderen Modulen, die nicht zur Standardbibliothek
gehören. Darunter ist z.B. pyExcelerator, welcher lesen und schreiben in Excel-Dateien
ermöglicht. Das verfolgte Ziel ist, dass sich alle gängigen Dateiformate mit den gleichen
Datenformaten arbeiten, und dabei das gleiche Ergebnis liefern.
